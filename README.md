# Cooler Monitor BR

Monitoramento de temperatura do computador em tempo real para Windows.

## Funcionalidades

- Monitoramento em tempo real de temperaturas de CPU, GPU, discos e placa mãe
- Gráficos dinâmicos de temperatura ao longo do tempo
- Log automático em CSV para análise histórica
- Ícone na bandeja do sistema
- Auto-inicialização com o Windows
- Interface moderna com tema escuro

## Tecnologias

- **.NET 9** + **WPF** (Windows Presentation Foundation)
- **LibreHardwareMonitorLib** — acesso a sensores de hardware
- **OxyPlot** — gráficos em tempo real
- **H.NotifyIcon** — ícone na bandeja do sistema
